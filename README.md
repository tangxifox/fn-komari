# Komari-fnOS

轻量级服务器监控工具，专注低资源占用与简单部署。  
本应用为 **fnOS** 第三方封装，使用官方二进制文件进行封装，支持 **x86_64** 与 **ARM64** 架构。

## 基本信息

| 项目           | 内容                                 |
|----------------|--------------------------------------|
| **默认访问端口** | `25774`                               |
| **支持架构**     | `x86_64`、`ARM64`                     |
| **项目地址**     | <https://github.com/komari-monitor/komari> |
| **主要特性**     | - 低资源占用<br>- 简单部署<br> |
